# Katkıda bulunma

Teşekkürler. Küçük düzeltmelerden büyük özelliklere kadar katkılarınızı memnuniyetle karşılarız.

## Nasıl başlarım?

1. Depoyu çatallayın (fork) veya doğrudan klonlayın.
2. `npm run setup` ile bağımlılıkları kurun.
3. `backend/.env.example` dosyasını `backend/.env` olarak kopyalayıp düzenleyin.
4. `npm run dev` ile yerelde çalıştırın.
5. Değişiklik öncesi `npm run verify` çalışsın (mümkünse).

## Pull request

- Tek bir mantıksal değişiklik = tek PR tercih edilir.
- Commit mesajları açıklayıcı olsun (Türkçe veya İngilizce olabilir).
- UI değişikliklerinde mevcut Tailwind / bileşen stiline uyun.

## Davranış

Saygılı ve yapıcı iletişim. Taciz veya saldırganlık kabul edilmez.
